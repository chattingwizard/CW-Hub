Hiring Workflow v3

Daily gate system (5 days) + final interview on exam day.

For Hiring Manager (Azahara) — Internal Use
← Back to School
Workflow Summary
PREP (Discord + access setup) → DAY 1 Foundations training → DAY 2 Tools/Infloww training → DAY 3 Journey training → DAY 4 Advanced training → DAY 5 Go Live training → EXAM DAY (Interview + quiz + live exam) → PROBATION (14 days).

V1 Rule: Deadlines are communicated by HM in Discord and enforced manually in Admin Panel (Pass Day / Fail & Deactivate). No automatic time cutoff in platform yet.
Global Rules (V3)
1. One day = one module: every day has mandatory self-study first, then live training and assessment.
2. No completion, no training: if recruit has not completed the assigned module to 100% before training time, HM disqualifies manually.
3. Manual gate decisions: HM decides pass/fail in Admin Panel after each day.
4. Pass: unlock next day module. Fail: deactivate account immediately.
5. Voice mandatory: mics open in live sessions. Silence counts negatively.
6. Interview moved: no early interview; interview is now part of exam day.
Admin Panel V1 Controls: Open student → use Pass Day & Unlock Next or Fail & Deactivate. The panel also shows current day and section completion status.
PREP

Batch Setup + Discord Welcome

Objective

Register recruits in the school, set Day 1 expectations, and make deadline/time crystal clear before training starts.

HM Actions

  • Generate invite code in Admin Panel → Invites with batch group name and choose the correct workflow (V3 Diario for new cohorts, Legacy only for old active cohorts)
  • Send Discord welcome message with exact Day 1 training date/time
  • Require recruits to register + complete Foundations (100%) before Day 1 training starts

Welcome Message (copy-paste)

Welcome to Chatting Wizard training. Your Day 1 live training is: {date} at {time} Before that time, you must complete ALL of this: 1) Register in the School with this invite code: {INVITE_CODE} 2) Complete the Foundations day (100%), including all quizzes Important: - If your module is not 100% complete by training time, you are disqualified. - This process is daily. Each day you pass unlocks the next day. See you in training. — Chatting Wizard Hiring Team
DAY 1

Foundations Training + Assessment

Focus

Foundations only: rules, role clarity, safety basics, and understanding if recruit can apply fundamentals.

Live Session Format

  • Quick verbal checks on Foundations topics
  • Google mock chat simulation (no Infloww yet)
  • Evaluate instruction-following, communication, and basic quality

Decision

  • Pass: click Pass Day & Unlock Next in Admin (unlocks Day 2 / Tools)
  • Fail: click Fail & Deactivate
DAY 2

Tools & Infloww Training

Focus

First day using Infloww. Recruit must arrive with Tools module completed 100%.

Mandatory Walkthrough Videos (before session)

These videos are embedded in the school (Module 2 — Tools & Scripts → Infloww Walkthrough Library). Recruits watch them as part of their Day 2 preparation.

  • Messages Pro
  • Notifications and Chatting
  • Manage Inboxes
  • PPV Timeline
  • Vault Media
  • Mass Messages
  • Employee Metrics
  • Select Online Fans (optional)

Live Session Format

  • Screen-share walkthrough in Infloww (quick recap)
  • Hands-on navigation drills (inbox, fan insights, vault, PPV history)
  • Script/location tasks and quality spot checks
DAY 3

The Journey Training

Focus

Conversation flow mastery (rapport, teasing, sexting, aftercare, branch logic).

Live Session Format

  • Rapid verbal checks + role-play by journey phase
  • Applied script usage inside Infloww scenarios
  • Error correction in real time and pass/fail decision
DAY 4

Advanced Training

Focus

Objections, customs (TPDs), game sequences, multitasking, and operational discipline.

Live Session Format

  • Top objections drill (speed + script accuracy)
  • Custom sales simulation and pricing logic
  • Multitasking pressure scenario
DAY 5

Go Live Training + Exam Readiness

Focus

Final prep day to verify readiness for exam-day interview + full assessment.

HM Decision

  • Pass Day 5: recruit proceeds to exam day
  • Fail Day 5: deactivate
EXAM DAY

Interview + Final Assessment

Order of Evaluation

  • Part 1 — Interview (moved here): motivation, ownership, communication quality, professionalism
  • Part 2 — Written Quiz: school final quiz threshold
  • Part 3 — Live practical exam in Infloww: unassisted execution

Final Decision

  • Pass: move to probation setup and onboarding handoff
  • Fail: deactivate and remove from training pipeline
14 DAYS

Probation

Ownership

After exam pass, HM hands off to Chatter Manager / Team Leader for 14-day probation operations.

REF

Quick Admin Checklist (V1)

Daily Flow

  1. Post next-day prep message in Discord with exact start time
  2. Before session start: verify module is 100% complete
  3. Run live training + assessment for that day
  4. In Admin: Pass Day & Unlock Next OR Fail & Deactivate
  5. Repeat until Day 5, then run exam-day interview + assessment

Data Source of Truth

  • Discord: training schedule and communicated deadlines
  • School Admin Panel: progress, manual day gates, deactivation decisions
